shared_ptr<std::fstream> ucp::log::get_fstream ( ) [inline]

This function is used to pass heap allocated file pointer from thread to thread in daemon mode. Logger doesn't work otherwise since we use a static global instance to log and statics are stack based storage and threads have their own stack.
